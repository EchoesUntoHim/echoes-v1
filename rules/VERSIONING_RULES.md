# Versioning Rules

- **x.0.0 (Major)**: Overall structure or design overhaul (e.g., changing the entire layout).
- **0.x.0 (Minor)**: Functional changes within a page or a single menu item change (e.g., changing a menu label).
- **0.0.x (Patch)**: Simple error fixes or small UI tweaks (e.g., button design).

*Note: Each segment can range from 0 to 99.*
