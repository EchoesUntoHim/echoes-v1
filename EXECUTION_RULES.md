# 사용자 지시 전 확인 사항 (Execution Rules)

1. **지시사항 이외의 코드는 절대 건들지 말 것.**
2. **지시 명령을 하면, 이 파일을 먼저 로딩해서 확인 후 진행할 것.**
3. **수정, 실행, 고쳐, 이 외에 단어에는 코드 수정하지 말 것.**
4. **코드가 수정된 작업을 하면, 반드시 버젼규칙에 따라 버젼업 표시할 것.**
5. **버젼업이 되면 작업내용을 상세하게(원코드 - 수정코드 전체기록할 것) 기록할 것.**
6. **작업 중 오류가 발견되면 보고하고 대기할 것.**

## 버전 관리 규칙 (Reference)
- **x.0.0 (Major)**: 전체 구조 또는 디자인 개편.
- **0.x.0 (Minor)**: 페이지 내 기능 변화 또는 단일 메뉴 수정.
- **0.0.x (Patch)**: 단순 오류 수정 또는 작은 UI 트윅.
