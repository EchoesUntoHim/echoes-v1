# 작업 계획서 (v1.10.0) - 플랫폼별 메타데이터 최적화 및 UI 개편

## 1. 작업 개요
유튜브와 틱톡의 플랫폼 특성에 맞는 페르소나를 분리하고, 메타데이터 생성 로직 및 UI 배치를 최적화합니다. 특히 틱톡의 단일 설명 필드 특성을 반영하여 통합 복사 기능을 구현합니다.

## 2. 주요 수정 사항
### 2.1. 데이터 구조 변경 (`src/types.ts`)
- `WorkflowState` 결과 인터페이스에 `tiktokMetadata` (fullContent 필드 포함) 추가.

### 2.2. AI 프로토콜 개편 (`src/App.tsx`)
- `generateYoutubeMetadata`를 `generatePlatformMetadata`로 확장.
- **유튜브 페르소나**: 검색 최적화 및 롱폼 가이드 중심.
- **틱톡 페르소나**: 바이럴 숏폼 전략가(FYP 중심).
- **출력 형식**: 틱톡은 캡션과 해시태그를 줄바꿈으로 통합한 단일 필드(`fullContent`)로 생성.

### 2.3. UI 레이아웃 개편 (`src/components/PublishTab.tsx`)
- 업로드 페이지 내 메타데이터 출력부를 2열 그리드로 분할.
- **좌측(유튜브)**: 제목, 설명, 태그 분리 출력.
- **우측(틱톡)**: 한번에 복사 가능한 통합 설명창 출력.
- 시각적 구분을 위한 플랫폼 테마 컬러(Red vs Primary) 적용.

## 3. 버전 정보
- **이전 버전**: 1.9.2
- **새 버전**: 1.10.0 (기능적 메뉴 수정 및 플랫폼 로직 분리)

## 4. 기대 효과
- 플랫폼별 특화된 카피라이팅으로 도달률 극대화.
- 단일 필드 통합으로 틱톡 업로드 시 편의성 증대.
- 직관적인 UI 배치를 통한 작업 효율 개선.
