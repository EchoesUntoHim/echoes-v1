# [v1.12.16] 정밀 자막 싱크(Timed Lyrics) UI 삭제 계획서

## 1. 개요
사용자님의 지시에 따라 현재 불필요하게 남아있는 "정밀 자막 싱크 (Timed Lyrics)" UI 블록을 화면에서 완전히 삭제하는 계획입니다.

## 2. 상세 수정 내용 (`src/components/LyricsTab.tsx`)
- 가사 탭 화면 하단에 렌더링되고 있는 `정밀 자막 싱크` 관련 전체 UI 블록(`GlassCard` 컴포넌트 포함, 약 120줄 분량)을 통째로 삭제합니다.
- 삭제되는 주요 내용:
  - "정밀 자막 싱크 (Timed Lyrics)" 타이틀 및 안내 문구
  - 시간, 한글 가사, 영어 번역을 개별 수정하는 입력 폼 리스트
  - "+ 가사 줄 추가" 버튼
  - "싱크 데이터 복사" 버튼

## 3. 기대 효과
불필요한 기능이 화면을 차지하지 않게 되어, 화면 스크롤이 줄어들고 전체적인 UI가 훨씬 깔끔하고 쾌적해집니다.
