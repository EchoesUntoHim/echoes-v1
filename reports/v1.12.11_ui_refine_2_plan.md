# [v1.12.11] 2차 UI 공간 최적화 및 테마 완벽 통일 계획서

## 1. 개요
사용자의 추가 피드백을 반영하여, 좌측 곡 정보 영역의 세로 폭을 '보컬 타입' 칸 수준으로 더욱 압축하고, 메인 버튼과 '현재 선택됨' 요소의 색상을 깔끔하게 통일합니다.

## 2. 상세 수정 계획

### 2.1 좌측 영역 (곡 정보 설정) 세로폭 극한 축소
- 현행: 여전히 내부 띄어쓰기(`space-y-6`, `space-y-4`)와 패딩(`p-4`)이 남아 있어 우측(보컬 타입 등)보다 깁니다.
- **변경**: 전체 그룹핑의 간격을 `space-y-6`에서 `space-y-3`으로 줄이고, 내부 간격도 `space-y-1.5` 수준으로 극단적으로 줄여 보컬 타입 칸과 크기를 맞춥니다.

### 2.2 '가사 및 프롬프트 생성' 버튼 색상 통일
- 현행: 전체 배경이 완전히 밝은 형광 녹색(`bg-primary`)으로 너무 튀어 보입니다.
- **변경**: 주제(Topic) 칸의 정돈된 테마와 통일하기 위해, 테두리와 텍스트에만 포인트를 주는 방식(`bg-primary/10 border border-primary text-primary`)으로 깔끔하고 고급스럽게 변경합니다.

### 2.3 '현재 선택됨' 악기 색상 변경 (흰색 계열)
- 현행: `Primary(Green)` 테마로 적용되어 있음.
- **변경**: 악기 이름 뱃지를 **흰색 글씨 + 투명한 회색 배경**(`text-white bg-white/10 border border-white/20`)으로 변경하여 눈이 편안하고 깔끔하게 구분되도록 수정합니다.

### 2.4 버전 동기화
- `package.json` 및 `rules/APP_CODE_MAP.md`의 버전을 **v1.12.11**로 업데이트합니다.

## 3. 확인 절차
- 좌측 곡 정보 창의 높이가 우측 보컬 타입 영역과 비슷하게 줄어들었는지 확인합니다.
- 메인 생성 버튼이 형광색 바탕에서 세련된 테마 색상으로 변경되었는지 확인합니다.
- 현재 선택된 악기 뱃지가 흰색 톤으로 깔끔하게 노출되는지 확인합니다.
