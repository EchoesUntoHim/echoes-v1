# v1.5.7-patch2 원곡 가사 정밀 싱크 매핑 수정 계획

## 1. 개요
원곡 가사 적용 시 분석된 시간(싱크) 정보가 틀어지는 문제를 해결하기 위해, 타임스탬프 보존 및 텍스트 매핑 로직을 정밀화합니다.

## 2. 주요 수정 사항

### [VideoTab.tsx]
*   **원곡 가사 정규화**: 입력된 원곡 가사에서 타임스탬프(`[00:00]`) 패턴을 모두 제거하여 순수 텍스트만 남깁니다.
*   **정밀 매핑 로직**:
    1. 분석된 가사에서 시간 정보(`[00:10]`)만 추출합니다.
    2. 추출된 시간에 원곡 가사의 해당 줄 텍스트를 결합합니다.
    3. 중복 타임스탬프 발생을 방지하여 싱크를 완벽히 보존합니다.
*   **유효성 검사**: 원곡 가사의 줄 수와 분석된 가사의 줄 수가 다를 경우에 대비한 폴백(Fallback) 로직을 보강합니다.
