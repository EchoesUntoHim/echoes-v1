# v1.5.7-patch3 가사 편집 레이아웃 최적화 및 싱크 최종 해결 계획

## 1. 개요
가사 대조 편의성을 위해 레이아웃을 개선하고, 원곡 가사 적용 시 발생하는 싱크 오류를 완전히 해결합니다.

## 2. 주요 수정 사항

### [레이아웃 재배치]
*   **대조 모드**: 상단에 '원곡 가사'와 '한글 가사'를 `grid-cols-2`로 배치하여 1:1 대조가 가능하게 합니다.
*   **영문 가사 분리**: '영어 가사' 입력란을 하단으로 이동시키고 전체 너비를 사용하도록 변경합니다.

### [싱크 매핑 로직 개선]
*   **정밀 인덱싱**: 분석된 가사에서 실제 시간이 포함된 라인만 필터링하여 원곡 가사의 라인과 정확히 매핑합니다.
*   **싱크 보존**: 분석된 타임스탬프(`[00:00]`)가 유실되거나 중복되지 않도록 결합 로직을 강화합니다.
