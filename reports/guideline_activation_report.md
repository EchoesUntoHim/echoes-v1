# 지침 준수 강제 가이드 활성화 및 결의 보고서

## 1. 개요
사용자의 지시에 따라 `rules/AI_STRICT_GUIDELINES.md`의 자기 제어 프롬프트를 활성화하고, `rules/EXECUTION_RULES.md`의 최신 12가지 규칙을 엄격히 준수할 것을 결의합니다.

## 2. 최신 지침 변경 사항 확인 (제3항 보강)
- **제3항**: 모든 지시사항은 반드시 **플랜(Plan)으로 보고**하고, 사용자의 **검토 및 승인**이 떨어진 후에만 진행해야 함을 확인했습니다.
- **제3-1항**: 기존의 '수정, 실행, 고쳐' 외에 **'승인'**이라는 단어가 포함되어야만 코드 수정 단계로 진입할 수 있음을 확인했습니다.

## 3. 향후 지침 준수 다짐
- 어떠한 상황에서도 "플랜 보고 -> 승인 -> 실행"의 3단계 프로세스를 건너뛰지 않겠습니다.
- 보고서 작성 시 12가지 규칙을 요약하지 않고 전체의 엄격함을 유지하겠습니다.
- 사용자의 "승인"이 있기 전까지는 어떠한 코드 수정 도구(multi_replace_file_content 등)도 사용하지 않겠습니다.

## 4. 지침 준수 선서
본 AI 어시스턴트는 사용자가 부여한 '절차적 정당성'을 존재의 최우선 가치로 삼으며, 단 한 번의 예외도 허용하지 않고 지침을 따를 것을 선서합니다.
