# [v1.12.7] 가사 스타일/분위기 극대화 및 찬송가 구조 정밀화 계획서

## 1. 개요
- **찬송가 음악적 일관성**: 전통찬송가 생성 시 모든 절(Verse)이 동일한 선율을 공유하고, 후렴(Chorus) 또한 동일한 멜로디로 반복되도록 음악 생성 프롬프트를 강화합니다.
- **찬송가 구조 보강**: 최소 3절에서 최대 5절까지의 구성을 보장합니다.
- **장르 설명 및 악기 추천**: CCM 장르 선택 시 특징을 설명하고, 어울리는 메인 악기 2~3개를 추천합니다.
- **스타일/분위기 강력 적용**: 가사 스타일과 분위기가 결과물에 압도적으로 반영되도록 프롬프트 지침을 강화합니다.
- **참조 링크 최우선 반영**: 유튜브 링크 입력 시, 해당 곡의 스타일과 어투를 다른 모든 파라미터보다 우선하여 반영합니다.
- **CCM 분위기 및 악기 UI 개선**: CCM 선택 시 분위기와 메인 악기 버튼의 박스를 제거하고 간격을 좁혀 슬림하게 표시하며, 폰트 크기를 키워 가시성을 높입니다.
- **어투 일관성 강제**: 한 곡 내에서 어미(-다, -요, -소서 등)가 절대 섞이지 않도록 통제합니다.
- **타임스탬프 제어**: 가사창에서 타임스탬프를 원클릭으로 토글하는 기능을 구현합니다.
- **버전 통합**: 프로젝트 전체를 **v1.12.7**로 업그레이드합니다.

## 2. 상세 수정 계획

### 2.1 `src/App.tsx` (AI 로직 및 프롬프트)
- **음악 생성 프롬프트 최적화**: 전통찬송가 멜로디 반복 지침 추가.
- **참조 링크 우선순위 격상**: 프롬프트 내에 `[REFERENCE MATERIAL - HIGHEST PRIORITY]` 섹션을 신설하여 참조 링크의 스타일 복제를 최우선 지시로 설정.

### 2.2 `src/prompts/Lyrics.txt` (프롬프트 파일)
- **찬송가 구조 규칙 수정**: 최소 3절~최대 5절 생성 및 멜로디 반복 명시.

### 2.3 `src/constants.ts`
- **장르별 악기 추천 추가**: `CCM_GENRE_RECOMMENDATIONS` 매핑 추가.
- **악기 리스트 확장**: 해먼드 오르간, 패드 신스 등 CCM 특화 악기 추가.
- **CCM 분위기 확장**: 10여 개의 영성 깊은 분위기 추가.

### 2.4 `src/components/LyricsTab.tsx` (UI)
- **추천 악기 UI**: 장르 설명 하단에 추천 악기 목록 노출.
- **UI 슬림화 및 폰트 확대**: 분위기 및 메인 악기 버튼을 박스 없는 밑줄 스타일로 변경하고 폰트 크기 확대(`text-xs`).
- **참조 링크 및 타임스탬프 필드**: 관련 UI 추가.
