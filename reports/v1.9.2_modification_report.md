# v1.9.2 상세 수정 보고서 (Detailed Diff Report)

**버전:** v1.9.2
**일시:** 2026-04-23
**작업 내용:** 가사 자막 폰트 개별화(한/영 분리) 및 스타일 다양화 (정자체 4종 + 귀여운체)

## 1. 개요
한글 가사와 영어 가사의 폰트를 각각 독립적으로 설정할 수 있도록 시스템을 분리하고, 각 언어별로 사용자 요청에 따른 정자체 4종 및 귀여운/감성 서체 목록을 드롭다운으로 재구성하였습니다.

## 2. 파일별 변경 사항

### 2.1 `src/types.ts`
- `TitleSettings` 인터페이스 확장:
  - `lyricsKoreanFont: string` 추가.
  - `lyricsEnglishFont: string` 추가.
- `createDefaultSettings` 오퍼레이션에서 두 필드의 기본값을 `sans-serif`로 설정.

### 2.2 `src/constants.ts`
- `LYRICS_KOREAN_FONTS`:
  - 정자체(4종): 본고딕, 나눔고딕, 나눔명조, 고운돋움.
  - 귀여운/감성체: 감자꽃, 하이멜로디, 개구체, 주아체, 푸어스토리, 나눔펜.
- `LYRICS_ENGLISH_FONTS`:
  - 정자체(4종): Inter, Roboto, Montserrat, Open Sans.
  - 감성/필기체: Dancing Script, Pacifico, Satisfy, Great Vibes, Cookie, Sacramento.

### 2.3 `src/index.css`
- Google Fonts 임포트 주소에 `Open Sans`, `Sacramento`, `Cookie` 등 새로 추가된 서체들을 포함하여 업데이트.

### 2.4 `src/components/VideoSettingsPanel.tsx`
- 가사 설정 UI 구조 변경:
  - '한글 자막 폰트'와 '영어 자막 폰트'를 각각 선택할 수 있는 독립적인 드롭다운 메뉴 배치.
  - `Globe` 아이콘 및 레이블을 활용하여 언어별 설정임을 명시.
  - 레이아웃 최적화를 통해 가독성 확보.

### 2.5 `src/components/VideoPlayer.tsx`
- 캔버스 렌더링 로직 고도화:
  - 가사 출력 시 `currentPair.kor`에는 `lyricsKoreanFont`를 적용.
  - `currentPair.eng`에는 `lyricsEnglishFont`를 적용.
  - 페이드 및 스크롤 모드 전체에서 언어별 폰트가 정확히 전환되도록 로직 수정.

### 2.6 버전 업데이트
- `package.json`: 1.9.1 -> 1.9.2
- `src/components/LandingPage.tsx`: v1.9.1 -> v1.9.2

## 3. 코드 비교 예시 (대표 사례)

**기존 가사 출력 로직:**
```typescript
const lFont = titleSettings.lyricsFont || 'sans-serif';
ctx.font = `bold ${canvas.width * (lyricsFontSize / 100)}px ${lFont}`;
```

**수정된 언어별 개별 폰트 로직:**
```typescript
const lKorFont = titleSettings.lyricsKoreanFont || 'sans-serif';
const lEngFont = titleSettings.lyricsEnglishFont || 'sans-serif';

if (currentPair.kor) {
  ctx.font = `bold ${canvas.width * (lyricsFontSize / 100)}px ${lKorFont}`;
  // ... fillText
}
if (currentPair.eng) {
  ctx.font = `bold ${canvas.width * (lyricsFontSize * 0.7 / 100)}px ${lEngFont}`;
  // ... fillText
}
```
