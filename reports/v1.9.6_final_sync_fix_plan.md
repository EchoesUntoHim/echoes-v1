# 최종 번역 동기화 수정 계획서 (v1.9.0)

## 1. 개요
시스템 안내 문구(플레이스홀더)를 가사로 오인하여 발생하는 중복 번역 및 데이터 오염 문제를 완전히 해결합니다.

## 2. 상세 수정 계획 (`src/App.tsx`)
- **수정 위치**: 552행 가사 자동 번역 `useEffect`
- **수정 내용**: 시스템 안내 문구 필터링 로직 추가
```javascript
    // [시스템 안내 문구 필터링]
    // 분석 중임을 알리는 플레이스홀더 텍스트는 번역 대상에서 제외함.
    const isPlaceholder = workflow.results.lyrics.includes("분석하고 있습니다") || 
                          workflow.results.lyrics.includes("Analyzing");
    if (isPlaceholder) return;
```

## 3. 검증 계획
- 음원 분석 시작 시 "자동 번역 시작" 로그가 뜨지 않는지 확인합니다.
- 분석 완료 후 고유의 한글 가사와 타임스탬프가 유지되는지 확인합니다.
