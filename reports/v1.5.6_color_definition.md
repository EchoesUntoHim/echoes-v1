# v1.5.6 UI 색상 정의 보고서

## 1. 개요
이미지 생성 탭 헤더에 배치될 액션 버튼들의 색상별 기능 정의입니다. 이는 사용자의 요청 사항(#10)에 기반합니다.

## 2. 색상별 기능 정의
- **Red (빨간색)**: 
    - 기능: 이미지 업로드 (Image Upload)
    - 역할: 사용자가 제작에 사용할 메인 배경 이미지를 직접 선택하여 업로드함.
- **Yellow (노란색)**: 
    - 기능: 음원 업로드 (Audio Upload)
    - 역할: AI 분석의 대상이 되는 오디오 파일을 업로드함.
- **Blue (파란색)**: 
    - 기능: 전체이미지 생성 (Overall Generation)
    - 역할: 모든 비주얼 에셋(메인, 틱톡, 쇼츠)을 AI를 통해 일괄 생성함.

## 3. 디자인 적용 방식
- 각 버튼은 해당 색상의 투명도가 적용된 배경(`bg-color-500/20`)과 테두리(`border-color-500/30`)를 가집니다.
- 호버 및 활성화 상태에서 색상의 선명도가 높아지도록 애니메이션을 적용할 예정입니다.
