# v1.12.1 수정 완료 보고서

본 보고서는 사용자 요청에 따른 가사 생성 페이지 및 영상 렌더링 페이지의 기능 개선 및 오류 수정 결과를 담고 있습니다.

## 1. 가사 생성 페이지 (LyricsTab.tsx) 수정 사항

### 1.1 DB 저장 내용 리스트 출력
- **구현**: `sunoTracks` 데이터를 활용하여 하단에 히스토리 섹션을 추가했습니다.
- **필터링**: 리스트 출력 시 가사(`lyrics`) 데이터가 존재하는 항목만 노출되도록 필터링했습니다.
- **페이지네이션**: 데이터가 많아질 경우를 대비해 20개 단위로 페이지네이션을 적용했습니다.

### 1.2 삭제 기능 추가
- **구현**: 각 히스토리 항목 옆에 삭제(`Trash2`) 버튼을 추가했습니다.
- **연동**: 삭제 시 `sunoTracks` 상태가 업데이트되며, 이는 `App.tsx`의 Effect를 통해 LocalStorage 및 Cloud DB(Firestore)와 자동으로 동기화됩니다.

### 1.3 타임스탬프 체크박스 추가
- **구현**: 한글 가사와 영어 가사 입력창 상단에 각각 타임스탬프 보기/숨기기 체크박스를 추가했습니다.
- **로직**: 체크박스 해제 시 `stripTimestamp` 유틸리티를 사용하여 `[00:00]` 패턴을 제거한 텍스트를 출력하며, 복사 시에도 이 상태가 반영됩니다.

## 2. 영상 렌더링 페이지 (VideoTab.tsx / VideoPlayer.tsx) 수정 사항

### 2.1 썸네일 이미지 표시 로직 수정
- **원인 분석**: `VideoPlayer`의 캔버스에서 이미지를 그릴 때 `crossOrigin="anonymous"` 설정이 `blob:` URL이나 일부 로컬 경로에서 로드 실패를 유발했습니다.
- **해결**: `VideoPlayer.tsx`에 이미지 로드 Fallback 로직을 추가했습니다. 1차 로드 실패 시 `crossOrigin` 설정을 제거하고 다시 시도하여 썸네일 이미지가 플레이어 배경에 반드시 표시되도록 개선했습니다.

## 3. 버전 관리 및 공통 사항
- **버전 업데이트**: `v1.12.0`에서 `v1.12.1`로 업데이트 되었습니다.
