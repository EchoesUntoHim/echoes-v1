# v1.9.4 상세 비교 보고서 (_detailed_diff.md)

## 1. 개요
v1.9.3에서 발생한 유튜브/틱톡 메타데이터 줄바꿈 누락 문제를 긴급 복구하고, AI 프롬프트 체계를 강화하여 플랫폼별 최적의 레이아웃을 보장합니다. 또한 절차 준수를 위해 버전을 1.9.4로 업데이트했습니다.

## 2. 주요 변경 사항

### 2.1. 버전 정보 업데이트 (`package.json`)
- **이전**: `"version": "1.9.3"`
- **현재**: `"version": "1.9.4"`

### 2.2. 유튜브 메타데이터 복구 (`src/App.tsx`)
v1.9.3에서 틱톡 수정을 시도하다가 파손된 유튜브 설명 줄바꿈 로직을 복구했습니다.
- **수정 내용**: 유튜브 설명 생성 시 섹션별로 명확한 줄바꿈(`\n\n`)과 이모지 활용을 다시 강조하여 가시성을 확보했습니다.

### 2.3. 틱톡 줄바꿈 레이아웃 강제화 (`src/App.tsx`)
단순한 지시를 넘어 **MANDATORY LAYOUT**과 **Example Layout**을 프롬프트에 포함하여 틱톡 업로드 칸에 즉시 붙여넣을 수 있는 최적의 상태를 보장합니다.
- **구조**: [후크 문장] -> [3줄 이상의 공백] -> [15개의 해시태그 무더기]
- **지시 내용**: 틱톡의 단일 설명창 특성을 반영하여 사용자가 수동으로 엔터를 칠 필요가 없도록 `\n\n\n\n` 삽입을 강제했습니다.

### 2.4. 전략 가이드 숨기기 기능 (유지 및 검증)
- v1.9.3에서 추가된 '2026 전략 가이드' 숨기기/보이기 기능이 정상 작동함을 확인했습니다.
- 이제 메타데이터 복사 시 가이드가 방해될 경우 우측 상단 `[X]` 버튼으로 숨길 수 있습니다.

## 3. 검증 결과
- `npm run build` 결과: **성공**
- 틱톡/유튜브 줄바꿈: 프롬프트 예시 적용으로 인해 더욱 견고한 레이아웃 보장.
