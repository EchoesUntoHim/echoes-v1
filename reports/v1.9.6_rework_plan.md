# 영어가사 한글번역 및 분석 로그 강화 계획 (v1.9.0 유지)

## 1. 개요
음원 업로드 시 영어 가사를 자동으로 한국어로 번역하고, 분석된 가사 데이터를 타임스탬프와 함께 터미널에 출력하는 기능을 구현합니다. 

## 2. 상세 수정 계획 (`src/App.tsx`)

### 2.1. AI 분석 프롬프트 강화
- **수정**: `analyzeAudioComprehensively` 함수의 프롬프트 내에 `언어 판별 및 번역` 섹션을 최우선 순위로 배치합니다.
- **지침**: 
    - 영어 곡일 경우 `lyrics` 필드에 **한국어 번역본**을, `englishLyrics` 필드에 **영어 원문**을 담도록 명시합니다.
    - 한국어 가사가 누락되지 않도록 "어떠한 경우에도 `lyrics` 필드에는 한국어 가사가 포함되어야 함"을 강조합니다.

### 2.2. 상태 동기화 로직 보완
- **수정**: 분석 결과 수신 시 `workflow.params.isEnglishSong` 상태를 AI가 판별한 결과(`result.isEnglish`)로 즉시 업데이트합니다.
- **효과**: 이후의 자동 번역 로직(`translateLyrics`)과의 충돌을 방지하고, UI 편집창에 올바른 언어 데이터가 로드되도록 보장합니다.

### 2.3. 터미널 로그 출력 강화
- **수정**: 분석 완료 시점에 `addLog`를 활용하여 가사의 모든 단락을 타임스탬프와 연결하여 출력합니다.
- **형식**: `[00:00] 가사 내용` (한 줄씩 순차 출력)

## 3. 버전 관리 및 준수 사항
- **버전 유지**: 사용자 지시에 따라 시스템 버전을 **v1.9.0**으로 고정합니다. (현재 임의 수정된 버전 표시가 있다면 v1.9.0으로 즉시 복구합니다.)
- **로직 보존**: 지시 사항 이외의 이미지 생성, 영상 렌더링 등의 기존 로직은 절대 수정하지 않습니다.
