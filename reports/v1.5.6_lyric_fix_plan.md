# v1.5.6 가사 번역 비용 최적화 및 실시간 반영 계획

## 1. 개요
불필요한 AI API 호출로 인한 비용 낭비를 막고, 가사 수정 사항이 비디오 플레이어에 즉시 반영되도록 로직을 개선합니다.

## 2. 주요 수정 사항

### [VideoTab.tsx]
*   **10초 지연 번역 (Debounce)**: 한글 가사 입력이 멈춘 후 **10초**가 지나야 영문 번역이 실행되도록 타임아웃을 조정합니다.
*   **초기 로딩 번역 금지**: 새로고침이나 컴포넌트 마운트 시, 기존 가사가 있을 경우 자동 번역이 트리거되지 않도록 `isFirstMount` 체크 로직을 추가합니다.
*   **영상 실시간 동기화**: 입력창의 `onChange` 이벤트가 비디오 플레이어에 전달되는 가사 상태를 즉각 업데이트하도록 보장합니다.
