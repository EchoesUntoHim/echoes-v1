# 영상 시작 시 밝아짐 현상 해결 계획서 (v1.9.0)

## 1. 개요
영상이 재생될 때 UI 오버레이의 투명도 전환 효과로 인해 화면이 서서히 밝아지는 현상을 제거하여, 시작부터 일정한 밝기를 유지하도록 수정합니다.

## 2. 상세 수정 계획 (`src/components/VideoPlayer.tsx`)
- **수정 위치**: 779행 중앙 플레이 버튼 오버레이 영역
- **수정 내용**: 
    - `transition-opacity` 제거.
    - `bg-black/40` (재생 전 어둡게 만들기) 제거 또는 즉시 전환으로 변경.
```javascript
    // [수정 예정 코드]
    "absolute inset-0 flex items-center justify-center pointer-events-none"
```

## 3. 검증 계획
- 재생 버튼 클릭 시 화면의 밝기가 변하지 않고 즉시 재생되는지 확인합니다.
