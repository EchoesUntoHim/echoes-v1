# 작업 계획서 (v1.9.3) - 틱톡 줄바꿈 최적화 및 업로드 전략 가이드 추가

## 1. 작업 개요
틱톡 업로드 시 발생하던 줄바꿈 누락 문제를 해결하고, 사용자가 요청한 '2026 업로드 최적화 전략' 안내 섹션을 추가합니다. 또한 규칙에 따라 버전 정보(1.9.3)를 갱신합니다.

## 2. 주요 수정 사항

### 2.1. 틱톡 줄바꿈 로직 강화 (`src/App.tsx`)
- AI 프롬프트에 틱톡 본문(Caption)과 해시태그 사이에 **3줄 이상의 명시적 빈 행(Newlines)**을 강제로 삽입하도록 지시를 강화합니다.
- 복사 시 줄바꿈이 보존되어 틱톡 스튜디오에 즉시 사용 가능하도록 출력 형식을 고정합니다.

### 2.2. '2026 업로드 최적화 전략' 섹션 추가 (`src/components/PublishTab.tsx`)
- 업로드 페이지 상단 또는 틱톡 섹션 인근에 요일별 최적 업로드 시각(롱폼/숏츠 구분)을 보여주는 전략 가이드 섹션을 추가합니다.
- 이미지에서 제시된 데이터(월: 06-07/19-21, 화: 17-18/19-21 등)를 UI에 시각화하여 배치합니다.

### 2.3. 버전 정보 업데이트 (`/package.json`, `/src/constants.ts` 등)
- 시스템 전반의 버전을 **1.9.3**으로 일괄 상향합니다.

## 3. 절차 준수 사항
- 모든 수정 사항은 본 계획서에 대한 **'승인'** 후 진행됩니다.
- 수정 완료 후 `reports/v1.9.3_detailed_diff.md` 파일을 생성하여 변경 전후 코드를 기록합니다.
- 최종 동작 확인 후 사용자의 요청 시 백업을 수행합니다.
