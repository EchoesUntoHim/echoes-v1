## 4. 상세 변경 내용 요약

### UI (ImageTab.tsx)
1. **빨간 네모 (Image Upload)**: 제목란 옆으로 이동. 드롭다운 대신 버튼 리스트로 메인/틱톡/숏츠 대상 선택 및 개별 업로드 기능 통합.
2. **노란 네모 (Audio Upload)**: 기존 이미지 업로드 자리에 음원 업로드 배치.
3. **파란 네모 (Global Generate)**: 하단에 큼직한 '전체 이미지 일괄 생성' 버튼 배치.
4. **숏츠 슬라이더**: 0~5 범위 설정 슬라이더 복구 및 체크박스 고유 ID 부여.

### 기능 (VideoPlayer.tsx)
1. **자막 동기화**: `lyrics` 배열과 `timedLyrics` 배열을 인덱스로 매핑하여 원문 수정 시 즉각 자막 변경.
2. **노이즈 해결**: 시간 변경 시 `Math.abs`로 감지 후 즉시 `audio.currentTime` 설정 및 볼륨 0에서 1로 300ms 램프업.
