# v1.5.3 오류 픽스 보고서

## 1. 개요
v1.5.2 작업 완료 후 발견된 `nextTime is not defined` 오류에 대한 원인 분석 및 수정 내역을 보고합니다.

## 2. 오류 원인 및 해결 (`VideoPlayer.tsx`)
- **오류 증상**: `npm run lint` 수행 중 `VideoPlayer.tsx`에서 `nextTime` 변수를 찾을 수 없다는 에러 발생.
- **원인 분석**: `nextTime` 변수가 `if (parsedLyrics.timedLines...)` 블록 스코프 내부에 선언(`const nextTime = ...`)되어 있어, 이후 `fade` 디스플레이 모드 로직에서 접근할 때 참조 오류가 발생했습니다.
- **해결 방안**: 
  - `nextTime` 변수의 선언부를 블록 바깥으로 이동(`let nextTime = totalDuration;`)하여 스코프를 확장했습니다.
  - 조건문 내에서는 선언 없이 변수에 값만 할당(`nextTime = lines[activeIdx + 1]?.time || totalDuration;`)하도록 수정하여 오류를 완벽히 해결했습니다.

## 3. 결과
- `npm run lint` 검사를 정상 통과(Exit code: 0)했습니다.
- 관련 파일의 버전 표기를 `v1.5.3`으로 업데이트했습니다.
