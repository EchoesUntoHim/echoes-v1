# v1.5.7 영상 렌더링 UI 개편 및 원곡 가사 반영 계획

## 1. 개요
*   **대상 버전**: v1.5.7
*   **주요 목적**: 영상 렌더링 탭의 레이아웃을 3:7 비율로 최적화하고, 사용자가 입력한 원곡 가사를 분석된 시간 정보와 결합하여 자막 정확도를 극대화합니다.

## 2. 주요 수정 사항

### [레이아웃 개편 (v1.5.7)]
*   **비율 조정**: 좌측(이미지/오디오 소스) 영역을 약 30%로 축소, 우측(가사 편집) 영역을 약 70%로 확장합니다.
*   **컴포넌트 배치**: 가사 입력란이 우측의 넓은 공간을 차지하도록 그리드 설정을 변경합니다.

### [원곡 가사 기능 (v1.5.7)]
*   **필드 추가**: 한글/영어 가사 옆에 '원곡 가사' 입력 전용 텍스트 영역을 추가합니다.
*   **매핑 로직**: 
    *   사용자가 원곡 가사를 입력한 경우, AI 분석 가사의 타임스탬프(`[00:00]`)만 추출합니다.
    *   추출된 타임라인에 원곡 가사의 각 라인을 순서대로 매핑합니다.
    *   **"분석 시간은 그대로 사용"** 원칙에 따라 싱크 정보는 유지하고 텍스트만 교체합니다.
