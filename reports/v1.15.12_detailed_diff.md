### 1. 로그 저장 로직 최적화 (App.tsx)

**수정 전 (L277):**
```tsx
useEffect(() => { localStorage.setItem('echoesuntohim_logs', JSON.stringify(logs)); }, [logs]);
```

**수정 후:**
```tsx
// v1.15.12: 로그 동기화에도 1000ms 디바운싱을 적용하여 대량 로그 발생 시 프리징 방지
useEffect(() => { 
  const timeoutId = setTimeout(() => {
    localStorage.setItem('echoesuntohim_logs', JSON.stringify(logs.slice(-500))); // 최신 500개만 저장하여 성능 유지
  }, 1000);
  return () => clearTimeout(timeoutId);
}, [logs]);
```

### 2. addLog 함수 최적화 (App.tsx)

**수정 전 (L215):**
```tsx
const addLog = (msg: string) => setLogs(prev => [...prev, msg].slice(-20));
```

**수정 후:**
```tsx
const addLog = useCallback((message: string) => {
  const timestamp = new Date().toLocaleTimeString();
  const logEntry = `[${timestamp}] ${message}`;
  setLogs(prev => [...prev.slice(-499), logEntry]); // 최신 500개 로그만 유지하여 메모리 부하 방지
}, []);
```
