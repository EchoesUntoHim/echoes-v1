# v1.12.1: 정밀자막 기능 제거 및 히스토리 리스트 정규화

사용자의 요청에 따라 불필요한 "정밀자막 싱크" 기능을 제거하고, 가사 히스토리 리스트에서 DB(Firestore)에 저장되지 않은 불필요한 항목들을 정리합니다.

## User Review Required

> [!IMPORTANT]
> **정밀자막 싱크 제거**: 이 작업 이후에는 가사 편집창 하단에 나타나던 타임스탬프 편집기(Timed Lyrics Editor)가 사라집니다. 가사 싱크는 가사 텍스트의 순서에 따라 자동으로 처리됩니다.
> **리스트 정리**: 가사 탭 하단 리스트에서 사용자가 직접 '저장' 버튼을 눌러 DB에 보관한 항목들만 표시되도록 변경됩니다. Suno에서 가져온 임시 목록은 리스트에서 제외됩니다.
