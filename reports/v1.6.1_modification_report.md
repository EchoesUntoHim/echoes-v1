# v1.6.1 수정 내역 보고서 (Modification Report)

## [v1.6.1] - 2026-04-23

### 1. 개요
영상 설정 패널에서 조정한 오디오 페이드 인/아웃(Fade In/Out) 값이 실시간으로 반영되지 않던 문제를 해결했습니다.

### 2. 세부 수정 내역

#### 2.1 src/components/VideoPlayer.tsx (의존성 배열 수정)

**수정 사유:** 
- `VideoPlayer` 컴포넌트 내부의 오디오 렌더링 루프를 담당하는 `useEffect`의 의존성 배열에 `fadeInDuration`과 `fadeOutDuration`이 누락되어 있어, 값이 변경되어도 내부 로직이 갱신되지 않는 문제 발생.

**원코드 (useEffect 의존성):**
```tsx
  }, [imageSrc, isPlaying, parsedLyrics, type, startTime, duration, onEnded, title, showTitle, titleSettings, koreanTitle, englishTitle, lyricsStartTime, lyricsScrollEnd, lyricsFontSize]);
```

**수정코드 (신규 의존성 추가):**
```tsx
  }, [imageSrc, isPlaying, parsedLyrics, type, startTime, duration, onEnded, title, showTitle, titleSettings, koreanTitle, englishTitle, lyricsStartTime, lyricsScrollEnd, lyricsFontSize, fadeInDuration, fadeOutDuration]);
```

### 3. 결과 및 확인
- 사용자가 패널에서 페이드 인 시간을 변경하면, 미리보기 및 다운로드 녹화 시 변경된 시간이 정확하게 오디오 볼륨 조절에 반영됩니다.
- 10초 이상의 긴 페이드 타임 설정 시에도 정상 동작함을 확인했습니다.
- `lint_applet` 및 `compile_applet` 테스트를 통과했습니다.
