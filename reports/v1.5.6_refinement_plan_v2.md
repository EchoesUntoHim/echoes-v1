# v1.5.6 UI 고도화 및 버그 수정 상세 작업 플랜 (v2)

## 1. 개요
사용자의 요청에 따라 v1.5.6 버전의 이미지 생성 UI를 최적화하고, 영상 렌더링 관련 버그(자막 동기화, 오디오 노이즈)를 수정합니다. 모든 작업은 `rules/EXECUTION_RULES.md`를 엄격히 준수하며, 사용자의 **'승인'** 후 수정을 진행합니다.

## 2. 상세 작업 항목

### [A] 이미지 생성 탭 UI 개편 (src/components/ImageTab.tsx)
- **헤더 액션 바 구현**:
    - [Red] 이미지 업로드 버튼: `id="header-img-upload-btn"`, `whileHover={{ scale: 1.05 }}` 적용.
    - [Yellow] 음원 업로드 버튼: `id="header-audio-upload-btn"`, 상태(workflow.results.audioFile)에 따라 색상 변화.
    - [Blue] 전체이미지 생성 버튼: `id="header-gen-all-btn"`, `shadow-[0_0_25px_rgba(59,130,246,0.4)]` 적용.
- **우측 패널 기능 복구**:
    - **숏츠 개수 슬라이더**: `id="shorts-count-slider"`, 0~5 범위 설정 가능하도록 복구.
    - **개별 선택 체크박스**: 1~5번 각각 고유 ID 부여 및 선택 시 애니메이션 추가.
    - **버튼 정리**: 불필요한 "숏츠 보강생성" 버튼 삭제 및 "선택한 숏츠 개별 생성" 버튼 단일화.

### [B] 영상 렌더링 시스템 수정 (src/components/VideoPlayer.tsx)
- **자막 실시간 동기화**:
    - `lyrics` props 변경 감지 시, `timedLyrics` 배열의 텍스트 필드를 인덱스 기준으로 교체하여 즉각 반영.
- **오디오 노이즈 제거**:
    - `startTime` 변경 시 오디오 `currentTime`을 정밀 동기화하고, 볼륨 램프업(setInterval 30ms) 로직을 통해 팝 노이즈 방지.

## 3. 지침 준수 사항 (12가지 규칙)
- 지침 제2항에 의거하여 작업 전 `EXECUTION_RULES.md` 로딩 완료.
- 지침 제3항에 의거하여 본 플랜 보고 후 사용자의 **'승인'** 대기.
- 지침 제4항에 의거하여 버전은 **v1.5.6 유지** (사용자 명시적 지시).
- 지침 제10항에 의거하여 본 플랜을 `reports/` 폴더에 파일로 저장.
