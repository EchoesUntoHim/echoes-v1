## 2. 상세 변경 내역 (Detailed Changes)

### 2.1 데이터 타입 및 상수 (src/types.ts, src/constants.ts)
- `TitleEffect`에서 애니메이션 요소를 분리하여 `TitleAnimation` 타입을 신설하였습니다.
- `TITLE_EFFECTS` 상수를 시각 스타일 위주로 재정비하고, `TITLE_ANIMATIONS` 상수를 새롭게 추가하였습니다.

### 2.2 설정 패널 (src/components/VideoSettingsPanel.tsx)
- 효과와 애니메이션을 각각 선택할 수 있도록 두 개의 드롭다운 메뉴로 분리 배치하였습니다.
- 애니메이션 드롭다운은 초록색 테두리로 강조하여 사용자가 쉽게 찾을 수 있도록 개선하였습니다.

### 2.3 렌더링 엔진 (src/components/VideoPlayer.tsx)
- **중합 적용 로직**: 애니메이션에 의한 위치/크기 변환(Transform)과 효과에 의한 시각 처리(Shadow, Glow 등)가 독립적으로 계산된 후 중첩되어 나타나도록 수정하였습니다.
- **NaN 방지**: 페이드 인/아웃 시간이 0일 경우 수학적 오류로 제목이 안 보이던 문제를 `fadeIn > 0` 조건문을 통해 해결하였습니다.
- **오버레이 제어**: `showTitleOverlay` 값이 `false`일 경우 렌더링을 즉시 중단하도록 로직을 추가하였습니다.
