### 2.2. 틱톡 줄바꿈 로직 개선 (`src/App.tsx`)
AI 프롬프트를 강화하여 캡션과 해시태그 사이에 명확한 공백을 삽입하도록 수정했습니다.

**[Original]**
```typescript
[TIKTOK TASK]
Persona: ${tiktokPersona}
Requirement: Generate a single unified content for TikTok's description field. 
It must include: 
1. A short, punchy caption (Catchy/Curiosity-triggering).
2. 2-3 lines of space.
3. Viral hashtags (15 tags).
Structure it so the user can copy and paste the entire thing into TikTok's single description box.
```

**[Modified]**
```typescript
[TIKTOK TASK]
Persona: ${tiktokPersona}
Requirement: Generate a single unified content for TikTok's description field. 
CRITICAL: Use multiple NEWLINES (at least 3-4 \n characters) between the main caption and the hashtags so they are clearly separated.
It must include: 
1. A short, punchy caption (Catchy/Curiosity-triggering).
2. AT LEAST 3 EMPTY LINES (use \n\n\n\n).
3. Viral hashtags (15 tags).
Structure it so the user can copy and paste the entire thing into TikTok's single description box without manually adding spaces.
```
