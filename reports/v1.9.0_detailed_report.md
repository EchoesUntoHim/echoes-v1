## 1. 작업 개요
영상 페이지 및 이미지 페이지의 사용자 편의 기능 강화 및 신규 기능 추가.

## 2. 주요 수정 사항

### 2.1 영상 페이지 (Video Tab)
- **영어 곡 가사 번역 기능**: `isEnglishSong` 설정이 활성화된 경우, 영어 가사를 한국어로 번역하는 기능을 추가했습니다.
- **정지 상태 제목 노출**: 영상 재생 전 또는 일시정지 상태에서 제목이 배경 위에 즉시 보이도록 수정했습니다.

### 2.2 이미지 페이지 (Image Tab)
- **원본 가사 입력란**: 상단에 싱크 맞춤용 원본 가사를 입력할 수 있는 `textarea`를 추가했습니다.
- **오디오 분석 고도화**: 입력된 원본 가사가 있을 경우, 오디오 분석 시 이를 참조하여 정확한 타임스탬프를 생성하도록 개선했습니다.
