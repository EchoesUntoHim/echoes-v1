# v1.5.7-patch1 원곡 가사 매핑 오류 수정 계획

## 1. 개요
`VideoTab.tsx` 수정 중 발생한 문법 오류(`...` 기호 삽입)를 제거하고, 원곡 가사 매핑 로직이 실제 영상에 반영되지 않던 문제를 해결합니다.

## 2. 주요 수정 사항

### [VideoTab.tsx]
*   **문법 오류 제거**: 실수로 삽입된 `...` 문자열을 삭제하여 스크립트 실행 환경을 정상화합니다.
*   **로직 정상화**: `finalVideoLyrics` 변수가 비디오 플레이어에 정확히 전달되도록 프롭스 연결 상태를 재점검합니다.
*   **매핑 유연성**: 원곡 가사의 줄 바꿈이나 공백이 분석된 시간 정보와 충돌하지 않도록 매핑 로직을 보강합니다.
