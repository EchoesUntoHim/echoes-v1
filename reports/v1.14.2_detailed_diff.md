## 2. 코드 변경 사항

### 2.1 AI 이미지 생성 실연동 (MeditationTab.tsx)
- **변경 전**: Unsplash 정적 이미지 링크 사용.
- **변경 후**: 
  - `Imagen 4.0` API 직접 연동.
  - **강력한 종교적 프롬프트**: "Traditional Christian sacred art, glowing cross, holy light, open Bible, NO watermelons" 등을 삽입하여 할루시네이션 원천 차단.

### 2.2 렌더링 서버 업그레이드 (render_server.ts)
- **Base64 지원**: `data:image/...` 형식의 Base64 데이터를 수신하여 즉시 Buffer로 변환 후 렌더링하도록 수정.
- **자막 폴백(Fallback)**: `timedLyrics` 정보가 없어도 `lyrics` 텍스트가 있으면 영상 전체 구간에 걸쳐 자막을 생성하도록 로직 보완.

### 2.3 시스템 버전 동기화
