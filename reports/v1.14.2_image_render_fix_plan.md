# 이미지 및 렌더링 엔진 최적화 계획서 (v1.14.2_image_render_fix_plan)

## 1. 개요
- **버전**: v1.14.1 -> v1.14.2
- **목적**: 1분 묵상 Factory 이미지 품질 개선 및 렌더링 오류 원천 차단.

## 2. 상세 수정 계획

### 2.1 이미지 생성 엔진 (MeditationTab.tsx)
- **문제**: 정적 이미지 링크 사용으로 인한 기독교 색채 부족 및 할루시네이션(수박 등) 위험.
- **해결**: 
  - `handleGenerateAllImages`에 `Imagen 4.0` API 연동.
  - **프롬프트 강화**: "Christian spiritual atmosphere, holy divine light, cross, bible, cinematic sacred photography, highly detailed, 9:16 aspect ratio" 등 명확한 종교적 키워드 강제 삽입.

### 2.2 렌더링 서버 (scripts/render_server.ts)
- **문제**: 
  1. AI 생성 이미지(Base64)를 Axios로 Get 하려다 발생하는 에러.
  2. `timedLyrics` 데이터 누락 시 자막 생성 실패.
- **해결**:
  - `render-shorts` API에서 이미지 데이터가 Base64(data:image/...)인 경우 직접 Buffer로 변환하여 저장하는 로직 추가.
  - `settings.lyrics`가 일반 텍스트로 올 경우 자동으로 전체 구간 자막으로 변환하는 폴백(Fallback) 로직 추가.

## 3. 작업 단계
1. `src/components/MeditationTab.tsx` AI 이미지 생성 로직 구현.
2. `scripts/render_server.ts` Base64 처리 및 자막 호환성 패치.
3. `rules/APP_CODE_MAP.md` 및 `package.json` 버전 업데이트.
