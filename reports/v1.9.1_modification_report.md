# v1.9.1 상세 수정 보고서 (Detailed Diff Report)

**버전:** v1.9.1
**일시:** 2026-04-23
**작업 내용:** 영상 자막(가사) 전용 폰트 및 색상 설정 기능 추가

## 1. 개요
영상 설정 패널에서 사용자가 자막의 폰트(정자체 위주)와 색상을 직접 선택할 수 있도록 기능을 확장하였습니다.

## 2. 파일별 변경 사항

### 2.1 `src/types.ts`
- `TitleSettings` 인터페이스에 `lyricsFont: string`, `lyricsColor: string` 필드 추가.
- `createDefaultSettings` 오퍼레이션에서 기본값을 `sans-serif`와 `#ffffff`로 설정.

### 2.2 `src/constants.ts`
- `LYRICS_FONTS` 상수 추가: 본고딕(Noto Sans KR), 나눔고딕, 나눔명조, 고운돋움, 도현체 등 가독성 가독성 높은 정자체 5종 및 영문 정자체 5종 포함.

### 2.3 `src/index.css`
- Google Fonts 임포트에 `Noto Sans KR`, `Inter`, `Roboto`, `Montserrat`, `Oswald`, `Poppins` 등 정자체 폰트 추가.

### 2.4 `src/components/VideoSettingsPanel.tsx`
- '가사 설정' 섹션을 재배치하여 폰트 선택 드롭다운과 컬러 피커 추가.
- `Palette` 아이콘 추가 및 UI 레이아웃 조정.

### 2.5 `src/components/VideoPlayer.tsx`
- 캔버스 렌더링 루프 내에서 가사 출력 시 `titleSettings.lyricsFont`와 `titleSettings.lyricsColor`를 적용하도록 수정.
- `hexToRgba` 헬퍼 함수를 추가하여 색상값에 투명도(opacity)를 적용 가능하게 함.

### 2.6 버전 업데이트
- `package.json`: 1.9.0 -> 1.9.1
- `src/components/LandingPage.tsx`: v1.9.0 -> v1.9.1

## 3. 코드 비교 예시 (대표 사례)

**기존 가사 폰트 설정:**
```typescript
ctx.font = `bold ${canvas.width * (lyricsFontSize / 100)}px sans-serif`;
```

**수정된 가사 폰트 설정:**
```typescript
const lFont = titleSettings.lyricsFont || 'sans-serif';
ctx.font = `bold ${canvas.width * (lyricsFontSize / 100)}px ${lFont}`;
```
