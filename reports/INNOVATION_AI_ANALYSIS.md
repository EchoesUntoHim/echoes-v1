# Innovation AI Analysis Report (for ccmpro Integration)

## 📌 개요
혁신AI(hyeoksinai.com)의 수익화 관련 핵심 기능들을 분석한 결과입니다. 향후 ccmpro 프로젝트의 '수익화' 탭 구현 시 기초 데이터로 사용합니다.

## 1. 핵심 기능 및 로직
### A. 수익화 발굴 AI
- **입력**: 13개 필드 (직업, 기술, 자본, 시간, 목표 수익, 선호 채널 등)
- **출력**: 4주 단계별 로드맵 (환경설정 -> 브랜딩 -> 상품기획 -> 런칭)
- **핵심**: 사용자의 현재 상황을 데이터화하여 즉각 실행 가능한 단기 계획 수립.

### B. 혁신 트렌드 분석 AI
- **입력**: 수익화 발굴 데이터(MD 파일) 또는 수동 입력(플랫폼, USP, 예산 등)
- **출력**: 5단계 전략 보고서 (데이터 모델링, MVP 개발, 테스트, 마케팅 전략 등)
- **핵심**: 아이디어의 시장 적합성을 검증하고 기술적/마케팅적 전략 도출.

### C. 혁신 수익화 캘린더 AI (90일 플랜)
- **입력**: 이전 단계의 전략 문서 및 사용자 요청사항
- **출력**: **90일 마스터 플랜 (Daily Tasks)**
- **특징**:
    - 매일 수행해야 할 구체적인 작업(Day 1 ~ Day 90) 생성.
    - 실시간 생성 프로그레스 바 UI.
    - DOCX 다운로드 기능 제공.

## 2. UI/UX 디자인 요소
- **Theme**: Dark Mode (Zinc/Black)
- **Point Color**: Amber (#fbbf24) for actions, Emerald for success.
- **Components**: 
    - 단계별 진행 상태를 보여주는 상단 Stepper/Progress Bar.
    - 파일 업로드용 Dash-border Dropzone.
    - Markdown 렌더링 기반의 깔끔한 결과물 화면.

## 3. ccmpro 이식 아이디어
- **CCM 특화**: '사역 규모', '찬양 장르', '유튜브 채널' 등 CCM 아티스트 맞춤형 입력 필드 구성.
- **음원 수익화**: 유튜브 쇼츠, 인스타 릴스 연동 및 음원 유통(DistroKid 등) 전략 포함.
- **AI 통합**: Gemini API를 활용하여 가사 분석과 수익화 플랜을 연동.
